## 👋 自己紹介

東京在住のITプロフェッショナルとして、**クラウドインフラおよびITサポート**へのキャリア転換を進めています。AWSシステムの構築・運用を独力で実践しており、**4言語**でビジネスコミュニケーションが可能です。

EC2・RDS・Nginx・GitHub Actions・Amazon Bedrock・SESを使用した在庫管理システムを独力で構築しました。現在、**AWS ソリューションアーキテクト アソシエイト（SAA-C03）**の取得に向けて準備中です。

## 🏗️ 注目プロジェクト

### [在庫管理システム](https://github.com/amanrai00/inventory-system)

> AWS上でインフラ・自動化・AI統合を含めて独力で構築・デプロイしたFlask + MySQLアプリケーション。

| レイヤー | 詳細 |
|---------|------|
| **コンピューティング** | EC2 (Ubuntu) · Nginxリバースプロキシ · systemdサービス管理 |
| **データベース** | RDS (MySQL) · スキーマ設計 · 在庫データ管理 |
| **CI/CD** | GitHub Actions · `main`へのプッシュで自動デプロイ |
| **AI機能** | Amazon Bedrock 需要予測 · 日英バイリンガル出力 |
| **アラート** | Amazon SES · 在庫不足メール通知の自動化 |
| **セキュリティ** | RBAC · 管理者デコレータ · デモアカウントのアクセス制御 |
| **UI** | 日英バイリンガル · セッションベースの言語切替 |
